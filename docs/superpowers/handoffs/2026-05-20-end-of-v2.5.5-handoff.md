## How to install v2.5.5

**This is the first release that uses the new "Check Now" flow** (no more `gh workflow run` for routine releases):

1. wp-admin → S&N → Dashboard → Maintenance → **"Check Now"** under "Check for Updates"
2. wp-admin → Updates → **"Update plugin"** for Signal & Noise Tools
3. Done. From v2.5.5 onward, every future release lands the same way.

If the Updates UI doesn't surface v2.5.5 after Check Now (transient race): one fallback is still available — `gh workflow run deploy.yml --repo juanlentino/signal-and-noise-tools --ref v2.5.5`.

## Verification of v2.5.5 (smoke test)

After install:
1. Press **⌘K** → desktop-mode AI Copilot overlay opens
2. Type something like *"force-check for updates"* → AI should dispatch `sn-cmd-force-check`, transients clear, toast appears
3. Type *"show me the plugin version"* → toast shows v2.5.5
4. Type *"go to the RSS tab"* → navigates to wp-admin → S&N → RSS
5. Type *"purge all caches"* → AI should NOT auto-execute (we opted-out destructive commands); should respond chat-style suggesting manual ⌘K → "purge caches"

If any step fails, the diagnostic is on desktop-mode's `ai.ask()` side; check browser console for `[desktop-mode]` log entries.

## Architectural state of the plugin

**The plugin now has 4 dispatch surfaces for SN actions, all converging on the same underlying impl functions:**

```
[ snt_*_impl() / snt_cmd_impl_*() pure functions ]
            ▲   ▲   ▲   ▲
            │   │   │   │
   ┌────────┘   │   │   └────────────────────────┐
   │            │   │                            │
   │            │   └──────────┐                 │
   │            │              │                 │
[ wp-admin    [ Legacy        [ WP 7.0          [ desktop-mode
  SN forms ]   /signal-noise/  Abilities API     Cmd+K Copilot
               v1/{cmd,ai}/* ] /wp-abilities/    via aiCallable
                               v1/abilities/.../  flag
                               run ]              (Phase 16 v2.5.5) ]
```

All four route through ONE impl function per operation. Single source of truth maintained.

**11 abilities registered** (Phase 14 + v2.5.0 additions):
- `purge-all-caches`, `regenerate-og-card`, `get-deploy-status`, `clear-template-overrides` (Phase 14)
- `force-check-updates`, `full-reset`, `list-template-overrides`, `get-rss-stats` (v2.5.0 new)
- `ai-generate-meta-description`, `ai-generate-og-card-title`, `ai-generate-excerpt` (v2.5.0 — AI features as abilities)

**5 ability categories:** `maintenance`, `content`, `diagnostics`, `updates`, `ai-generation`.
